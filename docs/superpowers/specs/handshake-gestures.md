# 🤖 Reachy Mini — Antenna Handshakes

Every finished code beeps 🔊.

## Icons

| Icon | Means |
|------|-------|
| ⬅️ / ➡️ | tilt that antenna the way the arrow points |
| • | keep that antenna still |
| 🥁 | knock the two antennas together |

Antennas apart = **external**, antennas toward each other = **internal**.

Read each code left to right, **one beat per column** ① ② ③. Inside a beat, do **L** and **R** together. Keep it snappy: a pause over 1 second resets.

## When it listens

- 😴 **Asleep** (limp): only **Wake** works.
- 🙂 **Awake**: all the other codes work.
