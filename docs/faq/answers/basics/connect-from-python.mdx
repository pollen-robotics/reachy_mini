To control Reachy Mini, you mainly use the <mark style="background-color: rgba(59, 176, 209, 0.1);">`ReachyMini`</mark> class from the <mark style="background-color: rgba(59, 176, 209, 0.1);">`reachy_mini`</mark> package:

```python
from reachy_mini import ReachyMini

mini = ReachyMini()
```

This connects to the Reachy Mini daemon (started via <mark style="background-color: rgba(59, 176, 209, 0.1);">`reachy-mini-daemon`</mark> or <mark style="background-color: rgba(59, 176, 209, 0.1);">`reachy_mini.daemon.app.main`</mark>) and initializes motors and sensors.

<strong>Recommended pattern:</strong> use a context manager to automatically handle connection and cleanup:

```python
from reachy_mini import ReachyMini

with ReachyMini() as mini:
    # your code here
    ...
```
