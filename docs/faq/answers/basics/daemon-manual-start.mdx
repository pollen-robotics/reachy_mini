Yes. All examples assume you have already started the Reachy Mini daemon:

- Either via command line: <mark style="background-color: rgba(59, 176, 209, 0.1);">`reachy-mini-daemon`</mark>
- Or via Python: <mark style="background-color: rgba(59, 176, 209, 0.1);">`reachy_mini.daemon.app.main`</mark>
The <mark style="background-color: rgba(59, 176, 209, 0.1);">`ReachyMini`</mark> instance will then connect to this daemon.
